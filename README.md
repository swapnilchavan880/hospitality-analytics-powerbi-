## 🏨 Hospitality Analytics Dashboard | Power BI  

### 🔥 Overview  
This project analyzes hospitality domain data to uncover insights on **revenue, bookings, occupancy, and cancellations** using an interactive Power BI dashboard.

### 🛠️ Tools & Technologies  
- Power BI  
- Power Query (ETL)  
- DAX (Measures & Calculations)  
- Data Modeling  

---

### ⚙️ What I Implemented  

#### 🔄 ETL Process  
- Extracted data from folder  
- Cleaned & transformed data  
- Removed unnecessary columns  

#### 🧩 Data Modeling  
- Built relationships using Date, Property ID, Room Category  
- Connected booking & aggregated tables  

#### 📐 DAX Calculations  
- Created calculated columns:  
  - Week Number using `WEEKNUM()`  
  - Day Type (Weekend/Weekday) using `WEEKDAY()`  

- Built key measures:  
  - Revenue  
  - Total Bookings  
  - Total Capacity  
  - Successful Bookings  

---

### 📊 Dashboard Features  
- Revenue displayed in Millions  
- Occupancy, Cancellation & Realisation % metrics  
- Filters: City, Room Type, Month, Week  
- KPI cards with conditional formatting  
- Data bars for revenue & rating  
- Tooltip-enabled line charts  

---

### 🚀 Outcome  
This dashboard helps stakeholders understand **hotel performance, occupancy trends, and revenue drivers**, enabling better operational and pricing decisions.
